HTML (HyperText Markup Language)

HTML is the standard markup language for creating web pages. It describes the structure of web content using various elements and tags.

HTML is used to define the structure and content of web pages, including headings, paragraphs, images, links, forms, tables, and more.

HTML is written in a plain text format, using tags to define the different elements of a web page. These tags are surrounded by angle brackets, and usually come in pairs, with the opening tag preceding the content and the closing tag following the content.

<html> <head> <title>Page Title</title> </head> <body> <h1>Heading</h1> <p>Paragraph of text</p> </body> </html>
  • HTML is used to define the structure and content of web pages.
  • HTML is written in a plain text format, using tags to define the different elements of a web page.
  • HTML is used to create web pages that can be viewed in a web browser.
CSS (Cascading Style Sheets)

CSS is used for styling web pages. It allows you to control the layout, colors, fonts, and other design aspects of your website.

CSS is used to separate the presentation of a document from its structure, which is defined in the HTML code.

CSS is made up of a set of rules, known as styles, which are applied to the HTML elements on a web page.

body { background-color: #f2f2f2; font-family: Arial, sans-serif; } h1 { color: #00698f; }
  • CSS is used to control the layout, colors, fonts, and other design aspects of a website.
  • CSS is used to separate the presentation of a document from its structure.
  • CSS is made up of a set of rules, known as styles, which are applied to the HTML elements on a web page.
JavaScript

JavaScript is a versatile programming language that enables dynamic and interactive web experiences. It's commonly used for client-side scripting.

JavaScript is used to add interactivity to web pages, such as responding to user input, animating elements, and updating content dynamically.

JavaScript is also used for server-side programming, game development, and mobile app development.

function greet(name) { console.log(`Hello, ${name}!`); } greet('John');
  • JavaScript is used to add interactivity to web pages.
  • JavaScript is used for client-side scripting.
  • JavaScript is also used for server-side programming, game development, and mobile app development.
